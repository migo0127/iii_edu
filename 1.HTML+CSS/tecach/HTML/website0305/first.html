NBA》威廉森超勇猛 詹皇賽後抱抱

湖人隊2日再挫鵜鶘隊

湖人隊2日再挫鵜鶘隊，不過鵜鶘菜鳥錫安威廉森(Zion Williamson)猛砍生涯新高35分(投16中12)、7籃板，
其勇猛讓湖人主將詹姆斯(LeBron James)也讚嘆。詹姆斯賽後與威廉森擊掌相擁，詹姆斯還說了很多話。

詹姆斯說了什麼呢？

詹姆斯說了什麼呢？他笑笑地回答：「你們了解我的，這是我的責任。我自認有義務把下一代的籃球變得比這一代更好。當你看到很多優秀的新人(列舉一些人名)，可以發現目前的聯盟很棒。我希望能把我的智慧傳承下去，因為這是我從8歲就開始接觸的運動。」

19歲的威廉森這場有多次精彩鏡頭，包括在禁區單吃「魔獸」霍華(Dwight Howard)，不過他還是無法一下子就超越35歲的詹姆斯。詹皇繳出34分的大三元，賽後兩人相擁的鏡頭成為媒體焦點，畢竟威廉森可是「老皇帝」喬丹(Michael Jordan)看中的未來之星，他與喬丹品牌簽約7年，外傳為7500萬美元的天價。

詹姆斯本來不認識威廉森，兩人場外毫無交集，這次短暫的交談讓他們拉近不少距離。有人批評詹姆斯對敵人太友善了，詹姆斯只是笑笑回應：「請轉告他們，讓他們來親我的屁股！」
