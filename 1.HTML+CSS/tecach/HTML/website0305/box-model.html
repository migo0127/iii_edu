大谷翔平春訓低潮8打數1安打 日媒點出跟打擊姿勢有關

洛杉磯天使隊大谷翔平日前接受採訪的照片，粗壯的手臂線條讓人相當驚艷，不過他要過一陣子才能夠以投手身分上場，目前仍以打者身分進行春訓熱身賽，但目前成績不太理想，累計8個打數僅有1支安打。

大谷翔平前場面對德州遊騎兵的比賽

大谷翔平前場面對德州遊騎兵的比賽，才打出1支帶有1分打點的安打，結束連6打席沒有安打的窘境，不過他在今日與芝加哥白襪的比賽2個打數都吞下3振，似乎狀況有些低迷。

不過日本媒體認為，這跟春訓階段大谷翔平正在調整打擊姿勢有關，他此前一共有3種打擊方式，在日本時他用的是「逆一本足打法」，初到大聯盟時則是為了因應更快的球速，所以改成「不落足打法」，而本季則是開始想要調整成右腳會抬起的打擊姿勢。

大谷翔平本人對此表示，「雖然還不確定在進入正式賽季後仍會維持這樣的打擊姿勢，但在春訓時仍會持續練習這樣擊球。」不過也會有人擔心，抬起右腳轉移重心的打擊方式，會不會對他的膝蓋造成負擔，不過相信球隊的訓練員會隨時做出應對。
