全聯買酒精「時間點」曝光驚呆網友　開店5分鐘賣完店員解答原因

受到新冠肺炎（COVID-19）疫情影響，現在不僅口罩難買，就連酒精也經常是一早就搶購一空的狀態。一名網友在臉書社團「我愛全聯好物老實說」PO文分享，自己早上7時39分就出門準備去排隊，讓其他網友看了忍不住留言直呼，「怪不得方才送小孩上課經過，全聯有一堆人在排隊等開門」、「得上班，無法去排，每天只能看到『本日酒精已售完』」。

快訊／川普宣布：未來30天禁止所有來自歐洲的旅行

美國影星湯姆漢克斯（Tom Hanks）11日表示，他和妻子麗塔威爾森（Rita Wilson）確認感染新冠肺炎；目前兩人正在澳洲拍攝新電影中。 現年63歲的湯姆漢克斯在聲明中表示，他和妻子麗塔人正在澳洲，「我們感覺有點累，像是感冒了，身體有些疼痛，麗塔覺得身體忽冷忽熱，輕微的發燒。」他說，為了做正確的事，就像目前全世界需要的那樣，他們做了檢測，最終結果為陽性。
